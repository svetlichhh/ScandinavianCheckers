# ScandinavianCheckers
Локальная двух-пользовательская аркадная игра по правилам скандинавских шашек (хнефатафл)
## Ход игры
 Король или Конунг, располагается в центре игрового поля на «троне», защитники крестом по 2 от трона к периферии. Нападающие располагаются группой по 4 по четырем сторонам игрового поля. По углам доски находятся 4 крепости, задача короля, как можно быстрее попасть в одну из них, а защитников ему в этом помочь.
Задача нападающих поймать короля, не дав дойти ему до крепости.
## Правила
Фишки ходят горизонтально и вертикально по всему игровому полю. За исключением трона и крепостей, которые занимать может только Король. В данной версии перескакивать через трон допускается. При этом перескакивать через фишки свои либо противника не разрешается.
Фишка считается захваченной, если ее зажимают с двух сторон по вертикали или горизонтали фишки противника.
Захват Короля происходит: если король находится на троне с 4-х сторон; если рядом с троном с трех сторон; а в поле как обычная фишка (с двух сторон, по вертикали или горизонтали).
Пустой трон и крепости считаются враждебными фишками и фишка зажатая между ними и враждебной фишкой считается захваченной.
